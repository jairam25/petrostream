# PetroStream Simulation Suite

PetroStream is a high-fidelity, industry-aligned engineering simulation platform designed for upstream and midstream technical workflows. Built with a modern glass-morphic UI, it provides detailed pedagogical modules across the entire oil and gas lifecycle.

## Key Features & Modules

### 1. Offshore Engineering & Platforms
- **12 Technical Phases**: From Fixed Structures and Floating Systems to Subsea Production and Decommissioning.
- **Safety & Arctic Operations**: Detailed modules on fire & gas systems, ESD, and harsh environment engineering.

### 2. Petroleum Geology & Basin Analysis
- **8 Technical Phases**: Covering Basin Analysis, Source Rock Maturation (Oil/Gas windows), Reservoir Diagenesis, and Sequence Stratigraphy.

### 3. Corrosion & Materials Engineering
- **7 Technical Phases**: Mechanisms (Pitting, MIC, SCC), CO2/H2S modeling, Material Selection (CRA), and Asset Integrity Management (RBI/FFS).

### 4. Metering, Allocation & Fiscal Measurement
- **6 Technical Phases**: Flow principles (Coriolis, Ultrasonic), LACT units, MPFM (Multiphase), and Fiscal/Custody Transfer standards (AGA/API).

### 5. AI & Automation in Drilling
- **5 Technical Phases**: Automated Drilling Systems, Real-time ROP/MSE Optimization, Automated Geosteering, and Digital Well Planning.

### 6. Exploration & Geophysics
- Advanced Geophysics, Geochemistry, and Geospatial field data mapping.

### 7. Reservoir & Production
- PVT Analysis, Well Testing, Material Balance, Decline Curve Analysis, and advanced EOR (Chemical, Gas, Thermal, Microbial).

## Tech Stack
- **Frontend**: React, TypeScript, Vite
- **Styling**: TailwindCSS, Lucide React (Icons), Framer Motion (Animations)
- **Architecture**: Modular glass-morphic tabbed navigation

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
